CSS Snippet Cheatsheet

A beautifully styled webpage is just a simple copy and paste away!

Flexbox Row

Use these properties to create a flexbox row layout.


.row {
    display:flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout.


.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout using CSS Grid.


.grid {
    display: grid;
    width: 100%;
    grid-template-colomns:
     repeat(12, 1fr);
}

Box Transistion Glow

Use Transition and box shadows to glow on hover.


.card .card-title {
    border-radius: 8px;
    transition: all .5s ease-in-out;
}

.card:hover, .card:hover .card-title{
    box-shadow: inset 0px 0px 8px rgba(232, 102, 236 1), 0 0 15px rgba(232, 102, 236, 1);
}

Linear Gradients

This will create a background linear gradient from top to bottom.


.linear-gradient-bg {
    background-image: linear-gradient(var(--first-color), var(--second-color));
}

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point.


.card-title {
    position: relative;
    margin-top: -20px;
}
